# Сложно сосредоточиться
Проект посвящён адаптивной вёрстке страницы с использованием CSS Grid и работе с визуальным оформлением текста.

## Описание проекта

Цель проекта — отработать навыки:
- адаптивной верстки
- семантической HTML-разметки
- стилизации страницы с помощью CSS
- использования CSS Grid
- работы с шрифтами и цветовой палитрой под dark и light темы
- работы с переменными

## Используемые технологии
- HTML5
- CSS3
- CSS Grid
- CSS Custom Properties
- Media Queries
- Git
